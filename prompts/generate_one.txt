다음 조건에 맞는 코딩 문제 1개를 생성하십시오:

목적:
학생들이 다음 주제들을 정확히 이해했는지를 평가하기 위함입니다.

주제 목록:
- 클래스와 객체
- 생성자 (__init__)
- 객체 변수와 클래스 변수 구분
- 메서드 정의 및 호출
- self 키워드 사용
- 캡슐화 (접근 제어: public, private)
- 상속 (단일/다중)
- 오버라이딩
- super() 호출
- 클래스 구조 설계

문제 조건:
- 각 문제는 현실적인 시나리오(예: 은행 계좌, 동물, 학생, 직원, 작업자 등)를 기반으로 구성할 것
- 학생이 클래스를 정의하고 인스턴스를 생성한 뒤, 메서드를 호출하거나 속성을 다뤄야 해결 가능하도록 구성할 것
- 문제는 표준 입력/출력을 요구하지 않고, 클래스를 정의하고 호출하는 방식으로 구성
- 난이도는 대학 1~2학년 수준에 적절하도록 설정

출력 형식 예시 :

"Employee 클래스를 상속받은 Manager 클래스를 정의하고, pay() 메서드를 오버라이딩하여 보너스를 포함한 급여를 계산하도록 하세요."
"Account 클래스를 정의하고, 잔액(balance) 필드는 private로 설정한 뒤, deposit() 및 get_balance() 메서드를 통해 접근하도록 구현하세요."
